emlError_t emlInit ( )

Initializes EML.

This function must be called before making any other EML call.

Return values
EML_SUCCESSThe library has been properly initialized
EML_ALREADY_INITIALIZEDThe library was already initialized
EML_NO_MEMORYInsufficient memory for initialization
EML_BAD_CONFIGA malformed config file was provided
EML_UNKNOWNInternal library error

emlError_t emlShutdown ( )

◆ emlStart()

emlError_t emlStart ( )

Begins an energy monitoring section on all available devices.

Note
Calls to emlStart can be nested. A single data collection thread will run for each device for the duration of the outermost section.
Warning
EML is not yet thread-safe. Taking measurements from multiple application threads simultaneously is not supported.
Return values
EML_SUCCESSMonitoring has been started
EML_NO_MEMORYInsufficient memory for monitoring
EML_NOT_INITIALIZEDThe library had not been initialized
EML_UNKNOWNInternal library error

◆ emlStop()

emlError_t emlStop ( emlData_t **  results)

Ends the current monitoring section on all available devices and returns consumption data.

Ends the section marked by the last call to emlStart, and stops data collection threads if all sections have been closed.

Writes an array of emlData_t * (one per available device) in the memory pointed by results.

Note
The number of devices, and thus the necessary size for results, can be determined through emlDeviceGetCount.
Warning
EML is not yet thread-safe. Taking measurements from multiple application threads simultaneously is not supported.
Parameters
[out]resultsAddress where data handles will be copied
Return values
EML_SUCCESSMonitoring has been stopped, and data returned
EML_NOT_INITIALIZEDThe library had not been initialized
EML_UNKNOWNInternal library error
