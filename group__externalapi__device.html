emlError_t emlDeviceStart ( const emlDevice_t *  device)

Begins an energy monitoring section on a specific device.

Note
Calls to emlDeviceStart can be nested. A single data collection thread will run for each device for the duration of the outermost section.
Warning
EML is not yet thread-safe. Taking measurements from multiple application threads simultaneously is not supported.
Parameters
[in]deviceTarget device
Return values
EML_SUCCESSMonitoring has been started
EML_NO_MEMORYInsufficient memory for monitoring
EML_NOT_INITIALIZEDThe library had not been initialized
EML_UNKNOWNInternal library error
emlError_t emlDeviceStop ( const emlDevice_t *  device,
emlData_t **  result 
)

Ends an energy monitoring section on a specific device and returns consumption data.

Ends the section marked by the last call to emlStart, stopping data collection if all sections have been closed.

Writes an emlData_t * handle in the memory pointed by result. The number of devices, and thus the necessary size for this buffer, can be determined through emlDeviceGetCount.

Warning
EML is not yet thread-safe. Taking measurements from multiple application threads simultaneously is not supported.
Parameters
[in]deviceTarget device
[out]resultAddress where data handles will be copied.
Return values
EML_SUCCESSMonitoring has been stopped, and data returned
EML_NOT_INITIALIZEDThe library had not been initialized
EML_UNKNOWNInternal library error
