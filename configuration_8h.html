char* emlConfigFind ( )

Returns the configuration filename.

Looks for a configuration file in the following paths, in order:

  1. $XDG_CONFIG_HOME/eml/config (if $XDG_CONFIG_HOME is set and not empty)
  2. $HOME/.config/eml/config
  3. /etc/eml/config

Only the first existing file is returned, even if it is malformed.

Note
Configuration files are not required, and can be empty. Default values will be used for most missing entries (except for values made necessary by existing entries, such as the hostname for a network PDU that was declared in the configuration).
